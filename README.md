# WhatsApp Lead Booster PRO

Plugin WordPress para capturar leads qualificados e abrir atendimento direto no WhatsApp.

## Funcionalidades da versão 1.1

- Shortcode `[whatsapp_lead_booster]`
- Compatível também com `[whatsapp_orcamento]`
- Painel de configurações
- Dashboard com métricas
- Total de leads
- Leads do dia
- Visualizações do formulário
- Cliques no WhatsApp
- Taxa de conversão
- Tabela de leads
- Exportação CSV
- Notificação por e-mail
- Evento Meta Pixel `Lead`
- Evento Google Ads/GA4 `generate_lead`
- Campos de qualificação: serviço, orçamento e urgência

## Como instalar

1. Acesse o WordPress.
2. Vá em Plugins > Adicionar novo > Enviar plugin.
3. Envie o arquivo ZIP.
4. Ative o plugin.
5. Acesse **WhatsApp Booster > Configurações**.
6. Configure o número do WhatsApp no formato `5512999999999`.
7. Cole o shortcode em uma página ou no Elementor:

`[whatsapp_lead_booster]`
